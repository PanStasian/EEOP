Кнопки панели инструментов

Откат - Отменяет последнее изменение в документе.
Правка - Открыть инструмент Редактор для таблиц и диаграмм. Некоторые опции редактирования доступны только при наличии соединения с сервером в Интернете.
Сохранить - Создает новую копию веб-отчета с сохраненными изменениями.
Печатать - Печатает текущий объект (в представлении объектов) или все объекты (в представлении страниц).
Представление страниц - Переключает веб-отчет на вывод всех таблиц и диаграмм на одной странице.
Представление объектов - Переключает веб-отчет на вывод по одной таблице или диаграмме за раз.

Соединение с сервером

Состояние соединения веб-отчета с сервером в Интернете выводится в правом верхнем углу веб-отчета.
Соединение с Интернетом не требуется для открытия веб-отчета. Сохраненная копия веб-отчета позволяет просматривать все диаграммы и таблицы, а также дает ограниченные возможности редактирования при отсутствии соединения с Интернетом.
Соединение веб-отчета с сервером в Интернете обеспечивает гораздо большие возможности редактирования таблиц и диаграмм.
  • Если автор указал сервер в Интернете при создании веб-отчета, этот веб-отчет будет при открытии автоматически пытаться соединиться с этим сервером.
  • Если веб-отчет не соединяется с сервером, щелкните по сообщению о состоянии сервера, чтобы открыть инструменты для повтора попытки соединения, попытки соединиться с другим сервером или ввода нового адреса сервера.
  • Для справки о добавлении расширенных средств управления на интернет-сервер перейдите к http://www.ibm.com/developerworks/spssdevcentral.
  • Если вы задаете новое соединение с сервером, предпочтительный формат - http://xxx.xxx.xxx.xxx:xxxx.
